Здесь инструменты для защиты переписки через национальный мессенджер Max

MAX-Firefox-collector - расширение для firefox для получения текущей переписки в MAX в json виде
MAX-Firefox-decryptor - расшифровка сообщений (MFC + Python скрипты из основного пакета)
MAX app - пересобранный MAX.apk с вырезаннымии лишними разрешениями, только базовые функции (НЕ ШИФРУЕТ ПЕРЕПИСКУ)
