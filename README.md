# crypto-bro
========================

Дисклеймер

Статья 23. Конституция Российской Федерации
1. Каждый имеет право на неприкосновенность частной жизни, личную и семейную тайну, защиту своей чести и доброго имени.
2. Каждый имеет право на тайну переписки, телефонных переговоров, почтовых, телеграфных и иных сообщений. Ограничение этого права допускается только на основании судебного решения.

Данные программы не нарушают законов РФ.
Программа ориентирована на защиту комерческой тайны, но может быть использована и физ. лицами.

========================

Пакет база:

encryptor_cpu_multi.py - шифрование и дешифрование файлов на одном и нескольких потоках процессора (для файлов больше 10 мегабайт)
encryptor_gpu_cuda.py - шифрование и дешифрование файлов с использованием ускорителя Nvidia
encryptor_gpu_vulkan.py - шифрование и дешифрование файлов с использованием Vulkan

Требования: 
Python способный в модуль cryptography
Большинство программ кроссплатформенные, но приоритетная платформа Windows 11 для предприятий редакции 25H2
