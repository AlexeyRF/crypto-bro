========================

Почему создаётся этот пакет программ \ расширений \ скриптов? <br>
Либералы устраивают тряску, что интернет в России закроют в марте 2026, я сомневаюсь, что это так, но чтобы иметь возможность вести защищённую переписку по любому каналу связи создаётся этот пакет. 

========================

Пакет база:<br>
encryptor_cpu_multi.py - шифрование и дешифрование файлов на одном и нескольких потоках процессора (для файлов больше 10 мегабайт)<br>
encryptor_gpu_cuda.py - шифрование и дешифрование файлов с использованием ускорителя Nvidia<br>
encryptor_gpu_vulkan.py - шифрование и дешифрование файлов с использованием Vulkan<br>

В папке dev старые версии программы

Требования: 
Python способный в модуль cryptography<br>
Большинство программ кроссплатформенные, но приоритетная платформа Windows 11 для предприятий редакции 25H2, Android 10, 11, 15 для смартфонов и WSA
