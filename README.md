Почему создаётся этот пакет программ \ расширений \ скриптов? <br>
Либералы устраивают тряску, что интернет в России закроют в марте 2026, я сомневаюсь, что это так, но чтобы иметь возможность вести защищённую переписку по любому каналу связи создаётся этот пакет. 

========================

Пакет база:<br>
crypto_rsa.py - шифрование и дешифрование сообщений и файлов по алгоритму RSA<br>
crypto_elip.py - шифрование и дешифрование сообщений и файлов по алгоритму элиптических кривых<br>
В релизах выложены скомпилированные версии программ для Windows, MacOS, RedOS

Пакет MAX:<br>
MAX.apk пересобранное приложение, с удалением всех разрешений кроме самых главных. За основу взята Rustore версия от 29.12.2025

В папке dev старые версии программ, всё сохраняю, кроме версий с критическими уязвимостями

Требования: 
Python способный в модуль cryptography<br>
Большинство программ кроссплатформенные, но приоритетная платформа Windows 11 для предприятий редакции 25H2<br>Android 10, 11, 15 для смартфонов и WSA
