Корень:<br>
crypto_rsa.py - шифрование и дешифрование сообщений и файлов по алгоритму RSA<br>
crypto_elip.py - шифрование и дешифрование сообщений и файлов по алгоритму элиптических кривых<br>
В релизах выложены скомпилированные версии программ для Windows, MacOS, RedOS<br>

Папка telecom:<br>
программы для шифрованной аудиовидеосвязи<br>
audio.py - клиент аудиосвязи по p2p<br>
video.py - клиент для передачи видео<br>
screen.py - клиент для передачи экрана<br>
voice.py - клиент для шифрования голоса для передачи информации через прочие каналы<br>
video_enc.py - клиент для шифрования видео для передачи информации через прочие каналы<br>
screen_enc.py - клиент для шифрования экрана для передачи информации через прочие каналы<br>

Папка hello:<br>
файлы для реализации windows hello api в python

Папка MAX:<br>
MAX_min.apk - пересобранное приложение, через android manifest удалены всех разрешений, кроме нужных для текстовых сообщений. За основу взята Rustore версия от 29.12.2025<br>
SCAM.apk - пересобранное приложение, через android manifest удалены излишние разрешения. За основу взята Rustore версия от 29.12.2025<br>
MAX_maker.exe - программа для создания своего мода на MAX<br>

Я не являюсь разработчиком Макса и лишь делаю модификации

В папке dev старые версии программ, всё сохраняю, кроме версий с критическими уязвимостями

Требования: 
Python способный в модуль cryptography<br>
Большинство программ кроссплатформенные, но приоритетная платформа Windows 11 для предприятий редакции 25H2<br>Android 10, 11, 15 для смартфонов и WSA
